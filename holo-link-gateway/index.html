Holo Link Gateway

Cette URL sert à convertir un lien HTTPS cliquable (Teams, Slack, etc.) en ouverture de l'app Holo via holo://.

Format attendu :

/open?h=holo%3A%2F%2FREPO%2Ffichier.md

Exemple :

/open?h=holo%3A%2F%2Fdocumentation%2FReadme.md
